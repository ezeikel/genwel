# Genwel Mobile App

Placeholder for the Genwel React Native mobile app.

## Planned Stack

- **Framework**: Expo SDK 54+
- **Navigation**: Expo Router
- **Styling**: NativeWind (Tailwind for React Native)
- **State**: React Query / Zustand
- **Authentication**: Clerk / Supabase Auth
- **Payments**: RevenueCat

## Development

Mobile app development will begin after the marketing site is complete.

### Future Commands

```bash
pnpm prebuild:ios    # Generate native iOS project
pnpm prebuild:android # Generate native Android project
pnpm ios             # Build and run on iOS simulator
pnpm android         # Build and run on Android emulator
pnpm start           # Start Metro bundler
```

## Features (Planned)

- Account aggregation via Open Banking
- Payday budgeting
- Bill tracking and reminders
- Savings goals and pots
- Family support mode
- Remittance tracking
- Financial education content
